The insert method belongs to the list type in Python and is used to add an item at a specific position in a list. It takes two arguments: the index where you want to insert, and the item to insert. This modifies the original list in place.

Basic Usage

You can insert at any valid index:

animals = ["cat", "dog", "bird"] # 0 1 2 animals.insert(0, "hamster") print(animals) animals.insert(2, "rabbit") print(animals) Note that when inserting a new item, all items at or after the specified index have their indices increased by one to accommodate the new item.

Using Negative Indices

You can use negative indices with insert():

my_list = [1, 2, 3, 4] my_list.insert(-1, 99) print(my_list)

Out of Bounds Indices

If you use an index equal to or greater than the list's length, the item is appended to the end:

my_list = [1, 2, 3] my_list.insert(10, 4) print(my_list)

But for adding to the end, the append is generally clearer and preferred.

Uses a negative index that is out of bounds will insert the new item at the beginning of the list: my_list = [1, 2, 3] my_list.insert(-10, 0) print(my_list)

Return Value

The insert method returns None, so you typically don't capture its return value in a variable.
