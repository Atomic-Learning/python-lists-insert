The insert method belongs to the list type in Python and is used to add an item at a specific position in a list. It takes two arguments: the index where you want to insert, and the item to insert.

Basic insert Usage

The syntax is: my_list.insert(index, item)

colors = ["red", "blue"] print(colors) colors.insert(1, "green") print(colors)

Inserting at Different Positions

You can insert at any valid index:

animals = ["cat", "dog", "bird"] # 0 1 2 animals.insert(0, "hamster") print(animals) animals.insert(2, "rabbit") print(animals)

Inserting at the End

If you use an index equal to or greater than the list's length, the item is appended to the end:

my_list = [1, 2, 3] my_list.insert(10, 4) print(my_list)

But for adding to the end, append is clearer and preferred.

insert Modifies the Original List

insert modifies the original list:

my_list = [1, 2, 3] my_list.insert(1, "new") print(my_list)

Return Value

The insert method returns None, so you typically don't capture its return value in a variable.

Using Negative Indices

You can use negative indices with insert():

my_list = [1, 2, 3, 4] my_list.insert(-1, 99) print(my_list)
